Bienvenidos a mi Portafolio Profesional

Soy Juan Daniel Ladino Hernandez. Me especializo en el desarrollo de aplicaciones web y móviles, combinando creatividad con tecnología para resolver problemas de manera eficiente. Me apasiona aprender nuevas herramientas, trabajar en equipo y seguir creciendo en el área de la Ingeniería en Tecnologías de la Información.

Mi frase favorita es: "Primero resuelvo, después codifico".

🌱 Proyecto Integrador: EcoRegistro

Periodo: Mayo 2025 - Julio 2025

⚙️ Tecnologías y desarrollo

  • 💻 Desarrollé la interfaz de usuario con Windows Forms para una experiencia intuitiva.
  • 🔄 Implementé la lógica de negocio para registrar donaciones y calcular puntos de bonificación.
  • 🗄️ Utilicé SQL Server como base de datos para almacenar usuarios y donaciones.
  • ✅ Apliqué pruebas para asegurar la funcionalidad y la usabilidad del sistema.
  • 👥 Colaboré con un equipo de 4 personas, usando metodologías ágiles (SCRUM) para la gestión.

🌍 Impacto

EcoRegistro fomenta el reciclaje mediante un sistema de bonificación en puntos, motivando la participación ciudadana y apoyando la reducción de residuos.
